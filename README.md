# TapVertical - 縦分割画像ジェネレーター for X

Xで映える縦長4分割画像を簡単作成できるWebアプリ

## 機能

- **分割数選択**: 3分割 / 4分割 / 5分割
- **アスペクト比調整**: 9:20〜9:50
- **分割位置手動調整**: スライダーで自由に調整
- **区切り線スタイル**: なし / 黒線 / 白線 / ドット / ハート / スター
- **X隙間シミュレート**: 実際の表示を考慮した自動カット
- **余白処理**: ブラー / 単色 / グラデーション
- **高画質出力**: WebP / JPEG / PNG (品質80-100%)
- **Xタイムライン風プレビュー**
- **ZIP一括ダウンロード**

## 使い方

1. 画像をドラッグ＆ドロップ
2. 分割数・スタイルを選択
3. ダウンロードしてXに投稿

## 技術スタック

- Vanilla JavaScript + Canvas API
- Tailwind CSS (CDN)
- JSZip (CDN)
